## Game title

**Yes Man**

## Short description (itch.io “short description” field)

A tiny cozy browser idle game about saying yes to increasingly silly prompts. Tap **YES**, stack **Cheer**, buy domino **Auto-Yesers**, unlock stamps, and optionally prestige for a fresh outlook. Play in the browser — no install required for the web build.

## Long description (optional body copy)

You are a professional agree-er. Every prompt is another chance to collect Cheer, unlock passive generators, and discover what happens when optimism meets absurdity.

- **Click and choose:** Say yes to tiered prompts with flavor text and Cheer bonuses.
- **Idle progress:** Domino Auto-Yesers tick Cheer while you watch (or while the tab is away — full offline progress, no FOMO timers).
- **Optional prestige:** Reset for a multiplier when you are ready — never required to keep playing.
- **Honest idle:** Costs and effects are visible upfront. No loot boxes, no missable dailies.

Early builds include a local-first **Playtest + feedback** hub. Testers opt in to export JSON; there is no hidden telemetry.

**Play (web):** https://subtiliorars-sys.github.io/yes-man/  
**Playtest landing:** https://subtiliorars-sys.github.io/yes-man/playtest.html
