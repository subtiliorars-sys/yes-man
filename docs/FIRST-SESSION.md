# First session — honesty brief

**Audience:** new players and playtesters  
**Purpose:** set expectations before the first 5 minutes (tutorial honesty / no dark patterns)

## What this game is

- A local browser idle game: you tap **YES**, buy generators, optionally prestige.
- Progress lives in **your browser** (local save). No account is required to play.

## What you will **not** see

- Countdown timers that punish you for leaving
- Streak-shaming or “come back or lose progress” pressure
- Pay-to-progress or loot-box framing
- Forced account / email gate to keep playing

Offline progress is applied when you return. Prestige is **optional**.

## Suggested first 3–5 minutes

1. Tap YES a few times; buy the cheapest generator when you can.
2. Open **Stamps** once something unlocks — milestones, not chores.
3. If a prompt card appears, read it; accepting is optional.
4. Use **Playtest + feedback** only if you want to leave notes (see `PLAYTESTING.md`).
